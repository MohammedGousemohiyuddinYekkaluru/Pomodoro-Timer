# Pomodoro Timer ⏱️

A simple Pomodoro Timer built with HTML, CSS, and JavaScript to help improve focus and productivity.

## Features
- 25-minute countdown timer
- Start, Stop, and Reset controls
- Clean and minimal UI
- Time display updates in real-time

## How to Use
1. Click **Start** to begin the timer.
2. Click **Stop** to pause the timer.
3. Click **Reset** to set the timer back to 25:00.

## Technologies Used
- HTML
- CSS
- JavaScript
